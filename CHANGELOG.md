# Changelog

## 1.0.0 - 2026-06-10

- Creada aplicación web informativa completa para **Floristería Mi Lindo Jardín**.
- Añadida parte pública: inicio, sobre la floristería, catálogo filtrable, detalle de producto y contacto.
- Añadido aviso claro de que no existe compra online, carrito, pagos, pedidos online, apuestas online ni compra de lotería online.
- Añadido tratamiento informativo del servicio presencial de **Loterías y Apuestas del Estado**.
- Añadido panel privado con login, logout, dashboard, CRUD de productos y gestión de información general del negocio.
- Añadida subida local de imágenes con validación de tipo y tamaño.
- Añadida base de datos SQLite con semillas de administrador, datos del negocio y productos de ejemplo.
- Añadido formulario de contacto con validación, honeypot, rate limit básico y envío SMTP o buzón local de desarrollo.
- Añadidos estilos responsive cálidos y visuales para móvil, tablet y escritorio.
- Documentado arranque local, variables de entorno, credenciales de administrador y pruebas del formulario.
